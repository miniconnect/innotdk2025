Generálj adatokat egyéni paraméterekkel

Grafikus felületen állíthatod össze egy oszlop tartalmát, felparaméterezve az értékkészletet, a keveredést, és más összetevőket. A tényleges beállítás is hasonló, csak éppen egy konfigurációs fájlon keresztül.

Futtass lekérdezéseket

Egy HoloDB konfigurációs fájlt szerkeszthetsz YAML formátumban, és azonnal SQL lekérdezéseket is futtathatsz a keletkező adatbázison. A teljes struktúrában igénybe veheted a beépített autocomplete funkciót.

Teszteld egy fejlesztői projektbe építve

Egy egyszerű REST API-t kiszolgáló Micronaut projekt belsejébe nézhetsz bele. Módosíthatod a kódot, és az API hívásával megfigyelheted az eredményt. A teljes adatbázist a HoloDB szolgálja ki on-the-fly.
